Part 1: The Piecewise Function

Now, when I said that we would find the parent to this function, I was partially lying (oops!). The fact is, this function has many parents. Although some piecewise functions do only have one parent, some have many. Sometimes, a piecewise function is made of 2 or more functions of the same parent. Other times, a piecewise function is made of 2 or more functions of different parents. The thing that defines a piecewise function is that it is composed of pieces of 2 or more functions!

With this knowledge, let’s split this piecewise function into its different parts. A good trick is to find any place the piecewise function changes direction angularly and splice it.

As you can tell, once we have the piecewise function split up based on its "parts", we can start to see the different functions it is made up of! Some of the function is linear while another part is a section of a quadratic!

Part 2: Evaluating Equations of Piecewise Functions

Here’s what a piecewise function equation looks like:

The basic components of a piecewise function are the:

  • f(x) or output
  • the curly braces
  • the equations
  • restrictions/conditions

The output of f(x) is just like any other function, it names the function and represents the output. The curly braces are there to show that these modular equations are all part of this one function. These components are the regular parent functions that you are familiar with but all squished into one function. The components are the parts that we spent splicing previously in part 1, just in equation forms. Now with these individual modular functions, we need to be able to tell them to stop existing at certain points, and we use restrictions in the domain (x) to accomplish this!

Here we can see this linear equation component of the piecewise function stops at x = 4, so in the section that defines the respective linear function, we state that the function only works if x ≤ 4. If there is a closed dot at that section, then it means the linear function hits x = 4, however, if it were an open dot, then the function does not hit x = 4, and thus the restriction would be x < 4

Part 5: Determining a Piecewise Function's Continuity

Sometimes, piecewise functions are not continuous...

These are the cases where a piecewise function is discontinuous:

  • The component is discontinuous in its own domain, like the reciprocal function
  • Two components of the piecewise function don't meet up at the same y value
  • The two components do meet up at the same y value but they both have an open dot where they meet up, meaning the value is unevaluable in the piecewise function

So to check for a piecewise function's continuity algebraically, you can check 3 things. You see whether or not each function is continuous in its own domain (part 0), whether or not the connection points or endpoints are existent in the piecewise function (part 2) and whether or not the piecewise function components meet up at the same x and y

On a graph, you can just see if the piecewise function is traceable without lifting a finger, but with equations, you will have to follow those 3 steps.

So here is just a quick simple example of figuring out whether or not a piecewise function is continuous.

Second step is to evaluate whether or not the x at their connection point is evaluable. The connection point is shown in their restrictions, so use those x coordinates to check. Here we can plug x = 0 for the piecewise function as a whole and evaluate that it does exist.

So now the final step is to check whether or not the piecewise functions actually meet up at the x coordinate in the same y coordinate. To do this, you just plug in the x endpoint value for both the first and second modular function within the piecewise. If both evaluate to the same y value, we know that the piecewise function components meetup and its continuous!

If the piecewise function were discontinuous, we would have to state at which x value it was discontinuous based on our calculations.
